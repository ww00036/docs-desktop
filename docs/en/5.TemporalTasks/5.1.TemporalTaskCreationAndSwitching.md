### Create Temporal Task
This software is managed from the project dimension. Multiple tasks can be created under each project. After entering a project, the details of the first task under the project will be displayed by default. If there are no tasks under the project, the interface for creating a task will be displayed.
Click the "New Task" button ![](../img/desktop-manual-en/1720004397770-de90b447-fa66-4078-a7dc-1648e2822bcf.webp) above the created project panel to create multiple tasks under this project.

![](../img/desktop-manual-en/1720004778457-3446bb55-10ec-4929-a8d9-658f7fd50807.webp)

### Switching Temporal Tasks
Switching temporal tasks is achieved through the task thumbnail control ![](../img/desktop-manual-en/1720004805165-c111aba1-45c9-4906-b964-6c8afa4bac93.webp) above the created project panel. All tasks of this project will be displayed here. Click to switch and display different temporal tasks.

![](../img/desktop-manual-en/1720004734900-3a66a046-0796-447a-ae71-8912e81c6ccd.webp)
