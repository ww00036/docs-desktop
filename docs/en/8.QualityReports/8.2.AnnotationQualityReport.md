You can switch to the annotation report on the left side of the results report page. Click the ![](../img/desktop-manual-en/1711694810533-dd5ba768-2acd-4a69-9a51-bc8950260d2d.webp) button in the upper right corner to download the report file in PDF format. The annotation report mainly includes the following parts:
### Task Overview
Mainly includes a thumbnail and basic information of the task.

![](../img/desktop-manual-en/1711694881538-9332e419-bc0b-4d40-9cb5-66df44ba318f.webp)

### Annotation List
Lists the point, line, and polygon annotations and their detailed information in a table format.

![](../img/desktop-manual-en/1711694934636-6dc6ba76-3e60-45f9-adfe-1ea94dcd6ec4.webp)

### Annotation Details
Lists the specific data and map screenshots for each annotation in detail.

![](../img/desktop-manual-en/1711694981671-790ac053-a939-4f85-8353-4ad5e0a3b61b.webp)
