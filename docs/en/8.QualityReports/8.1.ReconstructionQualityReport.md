Click "View Report" to view the reconstruction report and annotation report for this task. Click the ![](../img/desktop-manual-en/1711694810533-dd5ba768-2acd-4a69-9a51-bc8950260d2d.webp) button in the upper right corner to download the report file in PDF format. The reconstruction report includes the following parts:
### Task Overview
Displays basic information about the task and time statistics for aerial triangulation and reconstruction.

![](../img/desktop-manual-en/1727084258331-8dcc2120-81eb-4aa3-ba40-80525fe464e2.webp)

### Aerial Triangulation Quality Report
Mainly includes photo collection information, camera calibration information, and image POS error information. If the photos for this task do not contain POS information, there will be no image position error map.
+ Survey Area Overview and Coverage

+ Image Position

### Reconstruction Quality Report
Mainly includes the environment in which the task was executed, reconstruction parameters, and an overview of the generated DSM. If the task only generated 3D results, there will be no DSM results.

![](../img/desktop-manual-en/1727084359410-3cb6725e-ddf2-4faa-b86f-674501bb3bc6.webp)
