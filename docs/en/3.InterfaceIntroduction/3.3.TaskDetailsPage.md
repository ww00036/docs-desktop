1. Time-Series Task Reconstruction and Result Switching Display Panel
Click the New Task button ![](../img/desktop-manual-en/1719985838549-89b37a4b-0136-42af-80f4-e965965ef61d.webp) to create new time-series tasks. The creation process is the same as the previous task. Click the thumbnail button of different time-series tasks ![](../img/desktop-manual-en/1719985867483-e28886fe-2372-4461-84bb-bebd99414883.webp) to display the results of that time-series task.
2. Result List Panel
Multiple selections are allowed to control the display of 2D, 3D, and annotation results.
3. Task Parameters Panel
![](../img/desktop-manual-en/1746614424345-631c34c7-b3ab-42ec-80d7-6ee4b3742552.webp): Click to display the reconstruction parameters set when creating the task, including input data, reconstruction accuracy, result selection, etc.
4. Result Display Toolset
![](../img/desktop-manual-en/1710903557689-890accb5-9fe1-4b41-866b-37771a763fdc.webp): Switch Basemap. Click to switch the basemap to satellite map or standard map mode.
![](../img/desktop-manual-en/1710908994694-d86e3983-e4b0-492c-82d6-1cdcd7531c30.webp): 2D Results. Click to switch and display 2D model results.
![](../img/desktop-manual-en/1710908903053-3492a114-8142-48d0-a8da-c0c2a46e0082.webp): 3D Results. Click to switch and display 3D model results.
![](../img/desktop-manual-en/1710903646897-078c5d6e-5ea2-4646-b45f-3d8ed6bd8977.webp): Map Snapshot. Click to take a screenshot and output.
![](../img/desktop-manual-en/1710903710974-4d7a7d9f-76eb-4966-ab50-bcc7da227609.webp): Map Reset. Click to restore the map to the image GPS position.
![](../img/desktop-manual-en/1710903825277-0f35c89e-e315-4e1d-8d6a-7152f9fdc877.webp): Map Zoom In. Click to smoothly zoom in on the map.
![](../img/desktop-manual-en/1710903856197-c93f06f2-8a47-4dd0-940d-7003ae124654.webp): Map Zoom Out. Click to smoothly zoom out on the map.
![](../img/desktop-manual-en/1710906887139-d2adf0b8-f7ce-4504-9a75-b46dec745d86.webp): Click to export annotation results.
5. 3D Map Tools
![](../img/desktop-manual-en/1727076733893-7c9e49e2-890b-415e-98d7-7ea31bc6b08c.webp): Capture a snapshot from the current view as a thumbnail.
![](../img/desktop-manual-en/1727076793348-30011f55-3762-4ea3-ac2e-8fed14322608.webp): Save the current view as the initial state of the model.
6. Viewpoint Switching Tool
![](../img/desktop-manual-en/1749442875256-bdf25898-ba9e-48c0-a02e-0d89b6998a10.webp): Switch to top-down view.
![](../img/desktop-manual-en/1749442885721-302610fa-d907-4b14-b80c-6dbf456f5600.webp): Switch to side view.
7. Surveying and Annotation Toolset
Click any surveying and annotation tool under the toolset to perform surveying and annotation on 2D or 3D results.
![](../img/desktop-manual-en/1710908667942-2462e479-c8d1-49bb-a421-dd0447fd330a.webp): Click to annotate point features and measure coordinates on the results.
![](../img/desktop-manual-en/1710908700597-5ae81f2b-3322-40ca-badb-f5b6ca5c1af2.webp): Click to annotate line features and measure distances on the results.
![](../img/desktop-manual-en/1710908736208-b1a61d7d-667d-4de8-af6e-6fdf311fed1a.webp): Click to annotate area features and measure areas on the results.
8. Dual-Screen Comparison Tool
Click to compare 2D and 3D results of time-series tasks.
9. Task Result Directory
![](../img/desktop-manual-en/1710906915077-db9daa21-b491-4aff-a88b-572dde68c5c3.webp): Click to open the task result directory. Among them:
+ 2D: 2D results folder
    - geotiffs: Orthoimages (DOM), Digital Surface Models (DSM), etc.
    - dom_tiles: Orthoimage tile results
    - dsm_tiles: Digital Surface Model tile results
    - split_dom: Orthoimage split results (requires enabling the split output button)
+ 3D: 3D results folder
    - model-b3dm: 3dtiles model results folder
    - model-osgb: osgb model results folder
    - point-las: las point cloud results folder
    - point-pnts: pnts point cloud results folder
    - ......
+ AT: Aerial Triangulation results folder
10. Reconstruction Information Panel
Displays relevant information about task reconstruction, including task name, task data type and number of photos, result quality, data acquisition time, task reconstruction start and end time, processing status, etc.
11. 3D Model Cropping Tool
![](../img/desktop-manual-en/1710958165168-646f0ff2-b04a-4202-b0fb-13cb9a64fca8.webp): Click, and users can draw a polygon on the 3D results to crop the model. It should be noted that this tool is only effective in the software's Viewer.
12. View Report
![](../img/desktop-manual-en/1710907880821-e90ea011-38f3-47a6-af85-a542ee15693e.webp): Click to enter the result quality report interface, which provides reports related to task reconstruction (task information, aerial triangulation quality, reconstruction quality, etc.) and annotation reports, and provides users with a result quality report download tool ![](../img/desktop-manual-en/1710958919355-5ea522c4-d3e0-40ce-83c9-caf4c28d8cbb.webp).
13. Delete Task
![](../img/desktop-manual-en/1710907909896-d35150a1-69d3-401c-b5e0-985d5292862a.webp): Click to delete the task and completely remove the task from the working directory.
