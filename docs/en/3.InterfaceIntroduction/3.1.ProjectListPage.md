1. Project Search
![](../img/desktop-manual-en/1710834981413-5b8d7a6c-05cf-42e3-b6a6-d702670456e9.webp): You can enter the complete project name or keywords to search for projects.
2. Data Tag Filtering
![](../img/desktop-manual-en/1710835728903-cf7c1330-4a59-4d8f-b782-fe22af313a99.webp): Click the dropdown box to display projects containing specific tags. Tags are user-defined project attributes.
3. Project Sorting
![](../img/desktop-manual-en/1710836907017-9bc42268-0d98-485c-9b9a-c1663e575ad6.webp): Click the dropdown box to sort and display the project list by certain rules. Specifically, there are three sorting rules: by name, by creation time, and by update time.
4. Project Display
![](../img/desktop-manual-en/1710837257644-02006e72-cf1f-4ed8-b133-7612edff4ce0.webp): Click to select the project list display mode, specifically:
![](../img/desktop-manual-en/1710838065565-b6ab9145-2b4c-419f-966b-06c86fad3c5b.webp): Card mode, click to display all projects in card format, showing project name, orthophoto thumbnail, status, creation time, update time, tags, and other information on the card.
![](../img/desktop-manual-en/1710838096101-1e04848c-f3f3-429a-a4d9-54612f957dd6.webp): List mode, click to display all projects in list format, showing project name, orthophoto thumbnail, status, creation time, update time, tags, and other column headers.
![](../img/desktop-manual-en/1710838120331-de5154dc-4125-4924-a3fc-a6c7d738d91d.webp): Map mode, click to display all projects on satellite map or standard map according to their geographic coordinates.
5. Batch Operations and Project Import
![](../img/desktop-manual-en/1749438999684-d61a4711-a205-40f9-ad6d-619f5349dd33.webp): Batch operation button, enter the batch operation interface to delete or export projects in batch.
![](../img/desktop-manual-en/1749439007161-f7d27989-304c-447b-b7c8-c10eeb5ab413.webp): Project import button: Supports importing projects exported from MipMap software.
6. My Cloud Space
![](../img/desktop-manual-en/1710841187786-13988d90-9112-4d5f-a4b3-44f864415c4b.webp): Click to enter the reconstruction public cloud MipMap Cloud webpage.
7. Software Settings
![](../img/desktop-manual-en/1710841339924-b97a5c40-a472-4293-a659-214287c60877.webp): Click to enter the user settings panel, including software version information, license management, and general settings such as working directory:
![](../img/desktop-manual-en/1746521481103-242ffe90-0836-422f-8b30-89e0562201ee.webp): Set the working directory for project modeling and other preferences.
![](../img/desktop-manual-en/1710841825501-6ca7b589-8099-43d4-9070-94b5d10e7789.webp): Unbind or change license, display software features and usage period permissions.
![](../img/desktop-manual-en/1710841894082-38c8f499-5a0f-46bf-8ed2-6d73bee99526.webp): Display software version information, update software version.
8. Personal Space
![](../img/desktop-manual-en/1710841977415-b628cca9-4584-4b91-9871-fa4cf83034eb.webp): Display personal account, reconstruction public cloud MipMap Cloud link, and perform account logout operations.
9. View User Manual
10. Create New Project
![](../img/desktop-manual-en/1710842165095-4b13914a-6cea-4a76-a3f9-fcdc0e34ba3c.webp): Click to create a new reconstruction project, set the project name, and jump to the \<Project Tasks\> interface.
11. Project Card
Click to enter the project's output interface to view the project's 2D/3D model outputs and model reports, and perform mapping and annotation operations.
In addition to project name, 2D output thumbnail, and update time, the project card also includes:
![](../img/desktop-manual-en/1710858864261-92162a56-d5d9-443e-aad0-a418a99bfeaa.webp): Project status, including Waiting (waiting for reconstruction), Running (reconstructing), Stopped (reconstruction stopped), AT (only aerial triangulation), 2D (has 2D output), 3D (has 3D output), Error (reconstruction error).
![](../img/desktop-manual-en/1710859152019-ff9208c3-bb45-4a48-92d0-6b0efa367a75.webp): Project management, click to perform project copy, rename, tag management, and project deletion operations.
