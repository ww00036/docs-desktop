### Overlay Display Switching
After importing overlays, the line drawing will be displayed on the orthophoto and model. You can switch the display status through the overlay bar on the left ![](../img/desktop-manual-en/1747738150004-c08a720b-345d-490f-97de-bb9f2b92eec0.webp) or the display button on the right ![](../img/desktop-manual-en/1747738168293-9e3933ea-9613-4236-b43b-1b98b2a667c2.webp).

![](../img/desktop-manual-en/1747791307071-7031d10c-9617-4ef2-8be0-224cea2b9b01.webp)

### Overlay Editing
In the 2D image display interface, click the overlay layer on the left to modify the overlay layer name, transparency, and color. Line-type overlays support modifying the line width. The overlay color defaults to the drawing color.

![](../img/desktop-manual-en/1747738239147-7c7622b0-79ba-44f9-8402-5fe0da11525e.webp)

In the 3D model display interface, click the overlay layer on the left. In addition to modifying attributes such as name and color, you can set whether the overlay is draped on the ground. After draping, the overlay will be attached to the model surface.

![](../img/desktop-manual-en/1747791396912-ed0ef146-f867-4d08-9214-a408224cb60d.webp)

When the overlay drape button is off, you can set the overlay height. After entering the elevation height, the line drawing will be raised by the corresponding height.

![](../img/desktop-manual-en/1747791350703-e514b468-9220-4afe-8d12-03dcce1135bc.webp)
