Move the mouse over "Overlays" in the results menu panel on the left, and the operation menu for overlay layers will be displayed on the right.

![](../img/desktop-manual-en/1747791183450-3b243bc9-95f6-497c-a08d-c0f890a972bd.webp)

Click the ![](../img/desktop-manual-en/1747737749972-3e8ca4fb-e986-4e24-825d-f06f83911955.webp) button, and the upload overlay dialog box will pop up.

![](../img/desktop-manual-en/1747737773780-d65a0184-ba97-4396-8540-2473d8de9d9e.webp)

Select the overlay data to be imported and the coordinate system of the overlays to import overlays drawn by other software into the model. Overlay files currently support four formats: dwg, dxf, shp, and GeoJSON. The shp format supports automatic reading of the coordinate system, while other overlay formats require manual selection of the coordinate system.
