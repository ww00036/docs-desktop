### Create a New Project
Click **New Project** in the upper-right corner of the software. Set the project name, then confirm to enter the task interface.

![](../../img/desktop-manual-en/4.2/新建项目.webp)

### LiDAR Task

In the task parameters panel on the left, select **LiDAR** as the task type to switch to the LiDAR task parameter configuration interface.

![](../../img/desktop-manual-en/4.2/激光雷达任务.webp)
