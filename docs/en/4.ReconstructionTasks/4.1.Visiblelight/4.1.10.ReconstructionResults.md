### Result Display and Switching

![](../../img/desktop-manual-en/1727081977381-4c26d563-ef31-43ea-aa9c-5a64637653ec.webp)

After reconstruction is complete, the software will automatically display the reconstruction results. The left side will show the result layers for this task. Click the "checkbox" in front of the layer to switch the display of different layers. Click the "2D/3D" switch button below the map to switch between displaying 2D and 3D results.
In the 3D result display interface, you can switch perspectives. Click the ![](../../img/desktop-manual-en/1749449734050-23be1148-a029-423c-8cb5-6276afce910f.webp) button to switch to a top-down view. Click the ![](../../img/desktop-manual-en/1749449737522-5b14eb61-ba0e-4261-9a09-79e3f527b7e2.webp) button to switch to a side view.

![](../../img/desktop-manual-en/1749449680182-bdfed692-a238-4cd7-8cc5-fbc51e801d0c.webp)

![](../../img/desktop-manual-en/1749449782362-968d1659-8433-4f59-aedc-8435c56917c4.webp)

+ Digital Orthophoto Map (DOM)
You can switch layer visibility and transparency in the right panel.

![](../../img/desktop-manual-en/1727082101906-edbf5121-ffcb-45ce-84ed-92c6011d9cc4.webp)

+ Digital Surface Model (DSM)
You can switch layer visibility and transparency in the right panel. You can use the slider in the legend to switch the displayed height range.

![](../../img/desktop-manual-en/1727082170075-8c111d3a-63d0-4948-8d85-9c18117f7e72.webp)

+ 3D Mesh Model

![](../../img/desktop-manual-en/1727082323286-54dd72d9-98b2-4bb9-bae0-1c15055c8530.webp)

You can switch the display mode of the 3D model in the right panel: double-sided rendering and wireframe model.

![](../../img/desktop-manual-en/1727082375277-aef2ba36-7cf6-408f-82a1-5b3e366d9d94.webp)

You can click the ![](../../img/desktop-manual-en/1711540707900-aed8e0e1-c0ae-4c06-96d5-ac2a44283912.webp) button to manually draw a region of interest to crop the model and display only the part you are interested in.

![](../../img/desktop-manual-en/1720003428035-764e4e07-1905-4e5b-be55-12963dd002aa.webp)

![](../../img/desktop-manual-en/1720003451904-e48ca925-df3e-4c01-ba78-e185dbeda540.webp)

> Note: 1. The model cropping function only affects the display effect of the model and will not modify the original model file. You can click the "Restore" button to restore the original model. 2. Model cropping only supports convex polygons.
>

### Get Original Results
On the task details page, click the ![](../../img/desktop-manual-en/1711540890729-5e844929-69e0-47ed-a8ec-0514c5ce7498.webp) button on the right to open the task's working directory, where you can view the task results. The directory meanings are as follows:
+ 2D: 2D results folder
    - geotiffs: Orthophoto (DOM), Digital Surface Model (DSM), and other results
    - dom_tiles: Orthophoto tile results
    - gsm_tiles: Digital Surface Model tile results
    - split_dom: Orthophoto split results (requires enabling the split output button)
+ 3D: 3D results folder
    - model-b3dm: 3dtiles model results folder
    - model-osgb: osgb model results folder
    - point-las: las point cloud results folder
    - point-pnts: pnts point cloud results folder
    - ......
+ AT: Aerial triangulation results folder
