Some antivirus software, such as 360 Total Security and 360 Antivirus, may misidentify the software as a virus, preventing installation or quarantining some software files, causing reconstruction to fail immediately upon clicking. This is actually a false positive caused by our software encryption. Software downloaded from official channels ([https://na.mipmap3d.com/download](https://na.mipmap3d.com/download)) is safe to use!
If you encounter this situation, we have two solutions:
1. **If files have already been quarantined, restore the quarantined files (common quarantined files include mipmap_engine.dll)**
2. **Add the software installation directory to the antivirus software's trust list**
Below is a demonstration using `360 Total Security` as an example on how to restore quarantined files and add to the trust list:
## 1 Restore Quarantined Files (common quarantined file mipmap_engine.dll)
(1) Open 360 Total Security, click `Virus Scan`, then click `Quarantine`

![](../img/desktop-manual-en/1741780538770-58ec80c3-32f1-46a6-87c0-23d481d9fce6.webp)

(2) In the `Quarantined` and `Blocked` sections, find quarantined files with "mipmap" in their names, such as mipmap_desktop.exe shown in the figure below. Another common one is mipmap_engine.dll. Select the file and click `Restore Selected`

![](../img/desktop-manual-en/1741781231733-df74234f-af3b-42a5-9988-09d8e5ddfcf6.webp)

## 2 Add Software Installation Directory to Trust List
**(1) Open 360 Total Security, click** `Virus Scan`, then click** `Trust List`

![](../img/desktop-manual-en/1741780785338-3ee39e82-d7c6-40a5-9c23-c36bd428dc19.webp)

(2) On the `Trusted` page, click `Add Directory`

![](../img/desktop-manual-en/1741780846891-f222d90b-dd8f-4d95-a25d-0170e601ebf9.webp)

(3) In the pop-up directory selection page, select the MipMap Desktop installation directory and click OK to complete the addition.
(The default software installation directory is C:\Program Files\MipMap\MipMapDesktop)

![](../img/desktop-manual-en/1741780925001-9fb5c89e-72f8-4220-a07e-597142eb907d.webp)
