### Modify Basic Information of Annotation
Select the annotation to be edited in the left panel, and the right panel will display the basic information of the annotation. You can modify the name, color, etc., of the annotation in this panel.

![](../img/desktop-manual-en/1711693188481-c7b833c4-469a-468c-8d88-07a86ec7253d.webp)

### Modify Nodes of Annotation
Select the annotation to be marked, and the map will display all nodes of the annotation.

![](../img/desktop-manual-en/1711693417708-b538c81b-792b-46db-92cb-fcbe695c5e49.webp)

+ Adjust Node Position
Click to select the node to be edited, press and move the mouse to modify the position of the node. Release the mouse to complete the modification. The right panel will update the measurement information of the annotation.
+ Add Node
Click ![](../img/desktop-manual-en/1711693480629-8b0d6f44-63fa-484b-ac71-65ff31b19030.webp) between two nodes to add a node between them. Drag to adjust the position of the node.
+ Delete Node
Click the node to be deleted, press the "Delete" key on the keyboard, or move the mouse to the node and right-click to delete the selected node. After deletion, the measurement information on the right side will be updated synchronously.
### Delete Annotation
Move the mouse to the annotation layer list on the left side. The operation buttons for the annotation will be displayed behind the layer. Click the ![](../img/desktop-manual-en/1711693647187-a744ccf6-c3ed-42f6-9120-6b161cdab35e.webp) button to delete the corresponding annotation.

![](../img/desktop-manual-en/1711693675321-60166f75-6d0f-481e-b126-878033c17dd3.webp)
