Move the mouse over "Annotation" in the results menu panel on the left, and the operation menu for annotations will be displayed on the right.

![](../img/desktop-manual-en/1747790972076-23f725c7-9aec-4978-ac7c-f5c896fe7a5c.webp)

Click the ![](../img/desktop-manual-en/1711693919461-34228f9a-d140-4c11-93f9-9c60889690bd.webp) button, and the export annotation dialog box will pop up.

![](../img/desktop-manual-en/1747730032572-7c0c4745-5c43-422f-9176-ea8006f3f3a9.webp)

Select the data format, coordinate system, and export location for the annotations to export the annotation layer in the selected format for use on other platforms. The data format currently supports four formats: Shapefile, DXF, GeoJSON, and KML. The coordinate system is consistent with the output result coordinate system by default, and users can select the coordinate system as needed.

![](../img/desktop-manual-en/1711694071968-0909e7b8-a68d-4701-bcbe-bf2515ca64e1.webp)
